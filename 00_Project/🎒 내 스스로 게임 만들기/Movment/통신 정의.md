기본 통신 구조 
```go
// packet
type Packet struct {
	Type string
	Data any
}
```


### 통신 종류
##### client가 수신(서버가 전송) 하는 경우
- 새로운 사용자 추가 (newPlayer)
```go
// New Player dto
type NewPlayer struct {
	ID string
	System System
}	
```
System 구조체는 다음 참고 [[movement의 Entity 정의]]

- 지속적인 정보 전달 (Update)
```go
// Update dto
type Update struct {
	System System
}
```


##### Server가 수신하는 경우(Client 입력 이벤트 발생)
- w, a, s, d 입력이 된 경우
```go
type MovementInput struct {
	Direction string
}
```
